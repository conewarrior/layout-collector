Privacy Policy

Last updated: February 24, 2026

Layout Collector is a Chrome extension that captures screenshots and metadata of web pages to build a personal design reference gallery.

Data We Collect

  • Page metadata — URL, page title, meta description, Open Graph tags, and favicon URL of pages you explicitly choose to save.
  • Screenshots — A visible-area screenshot of the current tab, captured only when you click "Save".
  • Categories — The page purpose and layout type you select for each saved layout.

How Data Is Stored

All data is stored in a Supabase project (PostgreSQL database + object storage). Screenshots are uploaded to Supabase Storage. No data is sent to any other third-party service.

Data We Do NOT Collect

  • Browsing history or activity on pages you don't explicitly save
  • Personal information (name, email, account credentials)
  • Cookies or tracking identifiers
  • Analytics or telemetry data

Permissions

  • activeTab — To capture a screenshot and read metadata of the current tab when you click the extension icon.
  • scripting — To extract page metadata (title, OG tags, favicon) from the active tab.
  • storage — To store extension settings locally.
  • sidePanel — To display the saved layouts gallery in a side panel.

Data Sharing

We do not sell, share, or transfer your data to any third party. Data is only transmitted between the extension and the configured Supabase project.

Data Deletion

You can delete any saved layout and its screenshot at any time through the extension's side panel. Deletion is permanent and removes data from both the database and storage.

Changes to This Policy

We may update this policy from time to time. Changes will be reflected on this page with an updated date.

Contact

If you have questions about this privacy policy, please open an issue at github.com/conewarrior/layout-collector.
